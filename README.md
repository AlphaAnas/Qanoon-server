# Qanoon-Server
A one stop solution for lawyers

Qanoon server is legally compliant, safe and ethical AI that can draft legal documents like Bail Applications, Legal Notices, Suits, Applications, Affidavits and more... all this while keeping your data secure. It can cite laws, and draft the content in final formatted output that you can directly send for printing. All this and without sends your precious data to any other LLMs for training.
